# Portfolio — Shushrutha Reddy Borra (EN/DE)

This is a simple portfolio for **GitHub Pages**, with:
- **Language switcher** (English/German)
- **Project screenshots** (replace placeholders in `/assets`)

## Publish on GitHub Pages
1. Create a repo named: `YOURUSERNAME.github.io`
2. Upload everything from this folder.
3. Go to **Settings → Pages**:
   - Branch: `main`
   - Folder: `/root`
4. Your site will be live at: `https://YOURUSERNAME.github.io`

## Add screenshots
Replace these files with your real project screenshots (PNG/JPG is fine):
- `assets/project-1.svg`
- `assets/project-2.svg`
- `assets/project-3.svg`

Keep the filenames the same, or update the `<img src="...">` paths in `index.html`.

## Add your CV
Upload your CV PDF into the repo (e.g. `Shushrutha_Reddy_CV.pdf`) and update the “Download CV” button link in `index.html`.
